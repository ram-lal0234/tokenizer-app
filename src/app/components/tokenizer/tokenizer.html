How It Works

Special Tokens:

  • 0 = <PAD> (Padding)
  • 1 = <UNK> (Unknown)
  • 2 = <START> (Begin)
  • 3 = <END> (End)

Process:

  • • Text is split into words
  • • Each unique word gets a unique ID
  • • IDs start from 4 (after special tokens)
  • • Unknown words get ID 1 (<UNK>)
  • • Vocabulary is built automatically on first encode
